# Filerev

Filerev is an application that helps users clean up and organize their Google Drive.

## Features

Filerev offers a variety of tools to manage Google Drive files:

### Storage Usage Summary
- Shows a high-level overview of storage usage across Google Drive, Gmail, and Google Photos.
- Provides a detailed view of which folders and file types use the most storage space.

### Duplicate File Finder
- Detects files in Google Drive where the content is 100% identical.
- Allows users to preview duplicates and remove them individually or in bulk.
- Can find duplicate photos specifically.

### Bulk File Deleter
- Removes large batches of files across a Google Drive account with a single click.
- Can delete all files in predefined categories (e.g., duplicate files, large files, old files).
- Can be used with filters to delete a customized subset of files.

### Folder Size Analyzer
- Shows which folders consume the most storage space in Google Drive.
- Provides a tree-based view of folders with a graphical representation of storage space usage.
- Shows the files within each folder.

### Hidden File Finder
- Finds "orphaned" files in Google Drive that consume storage space but are not visible in any folder.
- Allows users to restore these files by moving them back into a folder or delete them.

### Empty File & Empty Folder Remover
- Finds and removes empty files and folders that clutter a Google Drive account.

### File Type Sorter
- Allows users to browse files in Google Drive by file type (MIME type) or file extension.

### Shared Drive Organizer
- Helps manage and organize files within Google Workspace Shared Drives.
- Can find duplicate files within Shared Drives.

### Old File Manager
- Identifies and manages old files in Google Drive.
- Allows users to archive or bulk delete files older than a specified date.

### Temporary File Cleaner
- Finds and removes temporary files (e.g., `.tmp`, `.temp`, `.old`) that are created by desktop applications and synced to Google Drive.

### Clear Google Drive
- Removes all files from a Google Drive account with a single click.

### Restore Files in Google Drive Trash
- Restores all files from the Google Drive Trash to their original locations with a single click.

### Shared File Viewer
- Manages files shared with a user in Google Drive.
- Can remove or bulk delete shared files.
- Can find duplicate shared files.
- Can view shared files by file extension and owner.

## Plans and Features

Filerev offers different subscription plans (Free, Basic, Standard, Premium) that vary in the number of scans, file operations, and available features.

- **Free**: Limited scans, moves, and deletions. Can scan "My Drive".
- **Basic**: More scans and operations. Adds bulk delete and filters. Can scan synced computers and "Shared with me".
- **Standard**: More scans and operations. Adds the ability to select specific folders to scan, the Storage Analyzer, and the ability to view empty folders.
- **Premium**: Unlimited scans and the most operations. Adds the ability to scan Shared Drives.
