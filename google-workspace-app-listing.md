Filerev® is the Google Drive™ cleaner and organizer that helps you find duplicate files, see folder sizes, locate hidden files, and reclaim storage space. Scan your Google Drive™, review what's taking up space, and bulk delete clutter in a few clicks.

🌟 WHY FILEREV

🔹 Find duplicate files in Google Drive™ with byte-level accuracy (not filename matching).
🔹 See folder sizes — something Google Drive™ doesn't show natively.
🔹 Bulk delete thousands of files at once, far beyond Google Drive™'s native ~100-file selection limit.
🔹 Scans are metadata-only — your files are never downloaded or copied.
🔹 Over 7 billion files scanned and 37 petabytes of storage organized for our users.

🛠️ FEATURES

🔹 Duplicate File Finder — 100% byte-identical content match, with thumbnails, folder, owner, and date.
🔹 Duplicate Photo Finder — find duplicate photos in Google Drive™ across jpg, png, heic, raw, and more.
🔹 File Deduplication / Dedup — bulk remove duplicates with "select oldest" or "select newest" helpers.
🔹 Storage Analyzer — charts of storage usage across Google Drive™, Gmail™, and Google Photos™.
🔹 Folder Size Analyzer — tree-based view of your largest folders, sorted biggest first.
🔹 Bulk Delete Tool — mass delete files or folders in any category, with confirmation.
🔹 Bulk Trash Restore — restore every file in your Google Drive™ trash at once, preserving folder structure.
🔹 Hidden File Finder — locate orphaned files that still use quota but aren't inside any visible folder.
🔹 Large File Viewer — quickly find and delete the biggest files wasting space.
🔹 Old File Manager — find files older than 1 year (or any age) for archival, retention, or cleanup.
🔹 Empty File Remover — delete zero-byte files and empty folders.
🔹 Temporary File Cleaner — remove .tmp, .temp, .old, and .download clutter synced from desktop apps.
🔹 Shared File Viewer — browse files shared with you, filter by person, and find duplicates across owners.
🔹 Shared Drive Organizer — scan and clean Google Workspace Shared Drives (Premium plan).
🔹 Browse Files by Type and Extension — documents, photos, audio, video, archive, or any specific extension.
🔹 Video Duration Summary — total video hours in your Google Drive™ account or any folder.
🔹 CSV Export — export a file list from any category or filter for auditing, compliance, or migration.
🔹 Powerful Filters — narrow by folder, owner, date, size, and file type before taking action.
🔹 Custom Views — save filter-based views as bookmarkable URLs (paid plans).
🔹 Flexible Scan Scope — scan your entire Google Drive™, specific folders, Shared Drives, Shared with Me, Synced Computers, or Hidden Files.
🔹 Clear Google Drive™ — one-click option to remove all files if you want to start fresh (recoverable from trash for 30 days).
🔹 Thumbnail previews with zoom on every file.
🔹 Android and iOS apps included — clean your Google Drive™ from any device.

🚩 GET STARTED

 1. Click Install above
 2. Open Filerev in the Google Drive™ sidebar
 3. Sign in and start your first scan

After the scan, Filerev sorts your files into categories so you can review clutter, apply filters, and bulk delete what you no longer need. Deleted files go to Google Drive™ trash and are recoverable for 30 days.
